# 🌐 Nethub Project

## 📌 Overview

Nethub is a web-based project developed as part of the IWP (Internet Web Programming) coursework. It provides a structured interface for browsing content like novels, music, and media through multiple interconnected web pages.

---

## 🚀 Features

* 📚 Browse novels and reading content
* 🎵 Organized media sections
* 🖼️ Image-based UI for better user experience
* 🌐 Multiple HTML pages with navigation
* 🎯 Simple and clean layout design

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Assets:** Images, media files

---

## 📂 Project Structure

```
Nethub/
│── index.html
│── *.html (multiple pages)
│── *.jpg / *.png (images)
│── MUSIC/        (ignored in repo)
│── NOVELS/       (ignored in repo)
```
